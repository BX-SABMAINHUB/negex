# Negex - Fusion entre Excel y Canva

Aplicación web ultraprofesional de diseño y edición visual con capacidades de hoja de cálculo.

## Instalación

```bash
pnpm install
cp .env.example .env.local  # Configurar variables de Firebase
pnpm run seed               # Insertar plantillas iniciales
pnpm dev                    # Iniciar en localhost:3000
